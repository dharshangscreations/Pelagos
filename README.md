# Pelagos — Your Sanctuary In The Ocean

A stunning portfolio website showcasing Pelagos with immersive design and smooth scrolling interactions.

## 🚀 Deployment

This site is deployed on GitHub Pages. Simply push your changes to the `main` branch and the site will automatically deploy.

### Setup Instructions

1. Create a new GitHub repository named `username.github.io` (replace `username` with your GitHub username)
2. Clone this repository locally
3. Push the contents to your GitHub repository:
   ```bash
   git remote add origin https://github.com/username/username.github.io.git
   git branch -M main
   git push -u origin main
   ```
4. Your site will be live at `https://username.github.io`
